# Data Structures & Algorithms (DSA) Implementation in Java

## Description
This project provides a comprehensive implementation of core **data structures** and **algorithms** in Java.  
It demonstrates a strong foundation in fundamental computer science concepts, including linear and non-linear data structures, algorithm design, and graph traversal techniques.

Developed as part of the **Data Structures & Algorithms course (4th semester, 2020)**, this project showcases practical applications of these concepts through well-structured Java code and example usage.


---

## Features

### Data Structures
- **Linked List** (singly)  
- **Stack** (array-based implementation)  
- **Queue** (circular array implementation)  
- **Binary Tree** (insertion and traversals: inorder, preorder, postorder, level-order)  
- **Graph** (adjacency list) with **BFS** and **DFS**

### Algorithms
- **Searching:** Linear Search, Binary Search  
- **Sorting:** Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort  

### Demo Program
- `Main.java` demonstrates all data structures and algorithms with sample usage and outputs.
