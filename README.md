# Data Structures & Algorithms (DSA) Implementation in Java

**Description**
This project implements core data structures and algorithms in Java.

- Data Structures: LinkedList, Stack, Queue, BinaryTree, Graph (adjacency list)
- Algorithms: Linear & Binary Search, Bubble/Selection/Insertion/Merge/Quick sorts, BFS & DFS for graphs, tree traversals
- `Main.java` demo program with examples
